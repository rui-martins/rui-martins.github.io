Illuminants

Puzzle solving game where you simultaneously control three characters.

The project

Illuminants was the first game I ever created alongside my college classmates for the final project of the first year.

It got a lot of attention from the teachers and that brought us the opportunity to showcase the game in various portuguese game conventions like Lisboa Games Week, Moche XL Games World and others.

Despite having almost no coding experience, design patterns or SOLID principles knowledge, we managed to deliver a project in four months that was fun to play while being challenging at the same time, and it was polished enough to get awarded the college game of the year prize!

I started programming a new version of the game for mobile with a complete code rework but there is no deadline since I just do it on my free time.

My roles

  • Gameplay programming
    • - Implemented some of the game mechanics alongside my project partner;
    - Reworked the whole movement system (on mobile version) to make it work with mobile swipes and snap the player to the grid when necessary.

  • Game design
    • - I was involved in the the game design for the levels' layout and order.
