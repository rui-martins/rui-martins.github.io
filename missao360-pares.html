Missão 360 - Jogo dos Pares

Find the pairs of animals in the forest and learn more about them.

The project

Missão 360 is an enviromental education project created by CELPA that reaches thousands of young students.

In a partnership with Easylab which handled the graphics part, I created the game from scratch in Unity and was the only developer that worked on it.

This project consists of an endless runner where you have to pick up the trash left in the forest. You need to switch your character's color to match the garbage recycling color while dodging obstacles by jumping.
The velocity starts slow and the trash objects have the respective bin colors, but as you progress, it gets progressively faster and the trash starts to appear with the original colors.
